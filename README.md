# InterviewPrep

**InterviewPrep** is an AI-powered mock interview platform that helps candidates prepare for technical and behavioral job interviews. Practice with real-world questions, get instant AI feedback, and track your progress over time.

---

## 🚀 Features

- **AI-Powered Mock Interviews** — Role-specific questions generated by AI
- **Voice & Video Support** — Real-time speech recognition for recording answers
- **Instant Feedback** — Scored, detailed feedback on every answer
- **Candidate Dashboard** — Track performance, upcoming interviews, and results
- **HR Dashboard** — Schedule interviews, manage candidates, view results
- **Role-Based Auth** — Separate flows for Candidates and HR professionals
- **Dark / Light Mode** — System-preference-aware theme toggle
- **Responsive Design** — Mobile, tablet, and desktop-ready

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend will be available at **http://localhost:5173**

### Backend

```bash
cd backend
npm install
```

Create a `.env` file:
```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=3001
```

```bash
npm run dev
```

### Evaluator Service

```bash
cd evaluator_service
# See evaluator_service/.env.example for required variables
npm install
npm run dev
```

## 🎨 Design System

The UI is built with **TailwindCSS** and a custom design system featuring:

- **Dual mode** — Light/dark mode with system-preference detection
- **Color palette** — Indigo/violet primary, with success/warning/destructive semantics
- **Typography** — Inter font (Google Fonts)
- **Components** — Cards (glass, gradient, hover), buttons (primary, outline, ghost), badges, alerts
- **Animations** — Float, glow-pulse, slide-up, shimmer, recording-ring
- **Glassmorphism** — Frosted glass effects on key UI surfaces
